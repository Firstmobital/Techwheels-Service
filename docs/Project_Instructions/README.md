# Project Instructions

This primary category stores instruction-level documents for contributors and coding agents.

## Purpose

Use this category for:

- contributor workflow instructions
- AI/agent instruction contracts
- release/playbook instructions that are not module-specific runbooks

## Structure

- `active/` (optional) for currently evolving instruction sets
- `reference/` (optional) for stable instruction contracts

This category is intentionally initialized now and can be expanded as instruction docs are formalized.
